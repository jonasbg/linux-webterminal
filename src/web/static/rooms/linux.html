Introduction from the Instructor

Welcome to Linux I, the first step in your DevSecOps journey. This course will introduce you to the Linux operating system, focusing on command-line operations that are essential for any security-conscious developer.

By the end of this course, you'll be comfortable navigating Linux file systems, managing files and permissions, and performing basic system operations - all skills that form the foundation of secure DevOps practices.
