# Linux I - Command Line Mystery

Solve a murder mystery using nothing but the command line.

## What you'll learn

- Navigate the Linux filesystem with `cd`, `ls`, `pwd`
- Search through files using `grep`, `head`, `tail`, `cat`
- Combine commands with pipes to filter and process data
- Think like an investigator - piece together clues from structured text files

## The challenge

A murder has been committed. The police have gathered evidence in a set of files. Your job is to sift through the witness statements, crime scene reports, and suspect lists to figure out **whodunit**.

Everything you need is in your home directory. Start with the `instructions` file and follow the trail.

## Who is this for?

Anyone new to the terminal. No prior Linux experience needed - just curiosity and a willingness to type commands instead of clicking buttons.
